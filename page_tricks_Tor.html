Ouvrir le fichier de configuration de Tor :

Pour forcer Tor à se connecter à un serveur (nœud) d'un pays particulier sous Ubuntu, vous devez modifier le fichier de configuration de Tor. Voici comment procéder : Ouvrez un terminal et éditez le fichier de configuration de Tor (`torrc`) avec un éditeur de texte. Par exemple, vous pouvez utiliser `nano` ou `gedit` : ```bash sudo nano /etc/tor/torrc ``` ou ```bash sudo gedit /etc/tor/torrc ```

Ajouter les directives de configuration :

Pour forcer Tor à utiliser des nœuds dans un pays spécifique, vous pouvez utiliser les options `ExitNodes`, `EntryNodes` et `StrictNodes`. Voici ce que vous devez ajouter : - **Pour spécifier les nœuds de sortie (c'est-à-dire le pays par lequel votre trafic sortira de Tor) :** ```bash ExitNodes {FR} ``` où `{FR}` est le code pays du pays que vous voulez (ici la France). Remplacez `{FR}` par le code pays désiré (par exemple `{DE}` pour l'Allemagne, `{US}` pour les États-Unis, etc.). - **Pour spécifier les nœuds d'entrée :** Vous pouvez également définir les nœuds d'entrée de la même manière avec `EntryNodes` : ```bash EntryNodes {FR} ``` - **Pour s'assurer que Tor respecte ces préférences :** Ajoutez la directive `StrictNodes` pour obliger Tor à n'utiliser que les nœuds spécifiés : ```bash StrictNodes 1 ``` Voici un exemple de ce que cela pourrait donner dans le fichier `torrc` : ```bash ExitNodes {FR} EntryNodes {FR} StrictNodes 1 ```

Enregistrer et quitter :

Si vous utilisez `nano`, appuyez sur `Ctrl+O` pour enregistrer les modifications, puis `Ctrl+X` pour quitter l'éditeur. <\div>

Redémarrer le service Tor :

Après avoir modifié le fichier de configuration, vous devez redémarrer Tor pour appliquer les changements : ```bash sudo systemctl restart tor ```

Vérifier que Tor utilise bien les nœuds du pays choisi :

Vous pouvez vérifier que Tor utilise bien des nœuds du pays choisi en consultant votre nouvelle adresse IP avec un service comme [whatismyip.com](https://www.whatismyip.com/) tout en étant connecté à Tor. Ces étapes devraient vous permettre de forcer Tor à se connecter à des nœuds spécifiques d'un pays particulier. En résumé, en configurant Firefox pour utiliser `127.0.0.1:9050`, tu forces ton navigateur à utiliser Tor pour tout son trafic, ce qui améliore ta confidentialité en ligne en masquant ton adresse IP et en cryptant ton trafic.
